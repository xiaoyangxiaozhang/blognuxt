# Design Context — BlogNuxt

## Target Audience
技术博客读者、开发者社区成员。熟悉 Web 界面，期望快速获取文章内容，同时欣赏有品质的 UI 设计。

## Use Cases
- 浏览技术文章（首页、归档、标签、分类）
- 阅读文章详情（Markdown 排版）
- 发布评论与互动
- 浏览动态/说说
- 查看博主信息与公告

## Brand Personality
- **极简克制** — 以内容为中心，装饰服务于功能
- **技术优雅** — 深色默认、精确排版、品牌青色点缀
- **安静自信** — 不喧哗、不浮夸，细节里见品质

## Design System
- **双主题**: midnight-blue (dark default) / blue-white (light)
- **品牌色**: cyan `#84dfff` (dark), blue `#2563eb` (light)
- **字体**: 系统无衬线 + 正文霞鹜文楷
- **排版**: clamp() 流式字体 / _prose.scss Markdown 排版系统
- **网格**: CSS Grid 2-3 列卡片布局
- **圆角**: 24-30px 卡片 / 999px 标签 / 16-18px 小元素
- **间距**: 22-30px 卡片间距 / 40-56px section 间距
- **动效**: ease-out-expo 缓动 / 150ms-400ms 过渡级 / prefers-reduced-motion
