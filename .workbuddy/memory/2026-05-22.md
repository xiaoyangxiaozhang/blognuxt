## 主题优化：蓝色还原 + 纯白文字 + 切换丝滑度

### 颜色调整
- 品牌色从 cyan (#84dfff/#67a8e4) 还原为纯蓝色系：
  - 暗色：`--brand-accent: #60a5fa` (blue-400)、`--brand-accent-hover: #3b82f6` (blue-500)
  - 亮色：`--brand-accent: #3b82f6` (blue-500)、`--brand-accent-hover: #2563eb` (blue-600)
- 文字改为纯白：`--home-text: #ffffff`、`--comment-input-text: #ffffff`
- 清理所有硬编码色值：Header.vue logo glow rgba(132,223,255) → rgba(96,165,250)、LoginDilog 输入框 focus ring → var(--brand-accent)

### 主题切换丝滑度
- `html` 全局过渡：`transition: background-color 0.35s, color 0.35s`
- View Transitions API 集成到 Header.vue `applyTheme()`：
  - 支持浏览器时使用 `document.startViewTransition()` 做交叉淡化
  - 不支持时回退到 CSS transition
- View Transition 伪元素自定义动画（theme-fade-in crossfade）
- `prefers-reduced-motion` 同时禁用 View Transition 动画
